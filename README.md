# Dự án Nghiên cứu: Đánh giá khả năng và mức độ chấp nhận AI Agent trong công việc

## Mục tiêu nghiên cứu
Kiểm tra xem AI Agent có khả năng đáp ứng công việc tương đương con người hay không, và đánh giá mức độ hài lòng của người lao động khi có AI Agent hỗ trợ trong công việc của họ.

## Cấu trúc thư mục
- `data/`: Chứa các file dữ liệu gốc (đã được copy vào đây gồm 4 file CSV).
- `scripts/`: Chứa các file code Python (Jupyter Notebook hoặc file .py) dùng để phân tích dữ liệu và vẽ biểu đồ.
- `docs/`: Chứa các báo cáo (reports), hình ảnh biểu đồ, các câu lệnh prompt và tài liệu liên quan đến dự án. (Xem [Tài liệu Dataflow](file:///C:/Users/NguyenNgocKhanh/Desktop/k/sandbox/ai_agent_research_project/docs/dataflow.md))

## Quản lý Yêu cầu (Prompts) và Tiến độ
Tất cả các prompt bạn yêu cầu tôi thực hiện, các insight tìm được và code phân tích sẽ được tôi tổ chức và lưu trữ tại thư mục này để bạn dễ dàng quản lý.
